NEXT STEPS
================================================================================

1. Add JUCE Submodule
   git submodule add -b master https://github.com/juce-framework/JUCE.git JUCE
   git submodule update --init --recursive

2. Create Plugin Source Code
   Add to Source/:
   - PluginProcessor.h
   - PluginProcessor.cpp
   - PluginEditor.h
   - PluginEditor.cpp

   See Docs/QUICK_START.md for example code.

3. Customize Plugin Metadata
   Edit CMakeLists.txt:
   - PLUGIN_NAME
   - PLUGIN_MANUFACTURER
   - PLUGIN_CODE (get unique 4-char code!)
   - PLUGIN_MANUFACTURER_CODE (get unique 4-char code!)

4. Build and Test
   ./Scripts/build.sh --release --test

5. Commit to Git
   git add .
   git commit -m "Add CMake build system and CI/CD"
   git push origin main

6. Enable GitHub Actions
   Push to GitHub and workflows will run automatically

7. Optional: Set Up Code Signing
   Add GitHub Secrets for macOS and Windows signing
   See Docs/SIGNING.md

================================================================================
QUICK COMMANDS
================================================================================

Build:
  ./Scripts/build.sh --release --test

Clean Build:
  ./Scripts/build.sh --clean --release

Debug Build:
  ./Scripts/build.sh --debug

Generate Xcode Project:
  cmake -B build -G Xcode
  open build/gFractor.xcodeproj

Generate Visual Studio Solution:
  cmake -B build -G "Visual Studio 17 2022"
  start build\gFractor.sln

Run Tests:
  ctest --test-dir build --output-on-failure

Create Release:
  git tag v1.0.0
  git push origin v1.0.0
